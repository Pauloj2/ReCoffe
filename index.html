Experimente o Sistema de Recomendação de Cafés

Com base no modelo treinado em KNIME utilizando 8 atributos sensoriais, o sistema classifica um novo café no cluster mais próximo e recomenda blends de perfil semelhante.

Acessar o Recomendador de Café

Como Funciona

O sistema utiliza o modelo K-Means treinado para classificar novos cafés com base em seus atributos sensoriais, identificando o cluster mais adequado e sugerindo cafés similares.

Benefícios

  • Descoberta de novos cafés com perfis similares aos preferidos
  • Recomendações baseadas em dados objetivos
  • Exploração guiada do universo dos cafés especiais
